# Ronaldo Riska — Portfolio Interattivo

Benvenuto nel repository del portfolio professionale di **Ronaldo Riska**, sviluppato come piattaforma interattiva per competenze in:

- 🧵 **Tessitura** – simulatori trama/ordito, WeaveCAD, calcolo note colore, Gantt ciclo tessile
- ✂️ **CAD Moda** – modellistica, cartamodelli, concept design, sketch enhancement
- 📦 **Supply Chain & Production** – dashboard KPI, make-or-buy, Kanban, business planning

Il sito è completamente responsive, con URL puliti (canonici) e ottimizzato per dispositivi mobili.

## 🛠️ Tecnologie utilizzate

- HTML5 semantico
- CSS3 con variabili CSS, layout grid/flex, media query mobile-first
- JavaScript vanilla (nessun framework) – routing client‑side, store locale, temi dinamici
- Interfacce interattive: canvas, slider, simulatore Kraljic, OEE, Bullwhip, …
- Font: Inter, DM Sans, Cormorant Garamond
