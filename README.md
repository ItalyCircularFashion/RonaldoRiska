# Ronaldo Riska Portfolio

Sito personale statico pronto per GitHub Pages.

## Struttura

- `index.html`: contenuto semantico del CV digitale
- `style.css`: design system modulare con CSS variables
- `script.js`: interazioni leggere e base per moduli futuri
- `assets/`: immagine profilo e CV PDF scaricabile

## Deploy su GitHub Pages

1. Crea un repository GitHub.
2. Carica questi file nella root del repository.
3. In GitHub vai su `Settings > Pages`.
4. Seleziona `Deploy from a branch`, branch `main`, folder `/root`.

## Estensioni previste

- Dashboard MRP/MPS per pianificazione tessile.
- Tool di qualita e tracciamento difetti.
- Mappa compliance ESG e certificazioni.
